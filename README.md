# Easy Eats

`Análise e Desenvolvimento de Sistemas`

`Desenvolvimento Aplicação Web-Backend`

`Eixo 2 - 01/2024`

O objetivo geral deste projeto é desenvolver uma aplicação para gerenciar pedidos de uma lanchonete, restaurante ou similares.
